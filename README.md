# 🌸 Çiçek Tanıyıcı AI

Kendi eğittiğim MobileNetV2 tabanlı çiçek sınıflandırma modeli.

## 🎯 Ne Yapıyor?
Fotoğraf yükle → AI çiçek türünü tahmin etsin!
5 farklı çiçek sınıfı: Daisy, Dandelion, Roses, Sunflowers, Tulips

## 🛠️ Kullanılan Teknolojiler
- Python, TensorFlow, Streamlit, Plotly
- Transfer Learning (MobileNetV2)
- ~3.600 görsel ile eğitildi

## 🚀 Kurulum
pip install tensorflow streamlit pillow plotly
python train.py
python -m streamlit run app.py
